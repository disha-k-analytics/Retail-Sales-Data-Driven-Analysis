# Case Study: Data-Driven Insights for Retail Performance

## 📌 Project Overview
This project applies the analytical frameworks and core methodologies from Course 2 of the Google Data Analytics Professional Certificate ("Ask Questions to Make Data-Driven Decisions"). The objective is to demonstrate structured thinking, the SMART questioning framework, spreadsheet proficiency, and stakeholder management strategies to solve real-world e-commerce challenges.

---

## 📑 Core Methodologies Applied

### 1. Business Problem & SMART Questioning Framework
Before analyzing data, a structured approach was used to convert a broad business problem into targeted, actionable queries.
*   **The Problem:** A retail e-commerce platform noticed a decline in quarterly conversion rates.
*   **SMART Framework Solution:**
    *   **Specific:** Identified drop-offs at the payment gateway page.
    *   **Measurable:** Tracked the exact percentage loss using checkout web traffic data.
    *   **Action-oriented:** Focused on providing actionable UX recommendations to technical teams.
    *   **Relevant:** Directly aligned with the organization's goal to recapture lost revenue.
    *   **Time-bound:** Analyzed customer journey datasets restricted to Q2 performance metrics.

### 2. Spreadsheet Analytics & Data Foundations
To prepare and structure raw transactional data, the following analytical processes were defined:
*   **Data Quality Assurance:** Applied sorting and filtering techniques to clean user feedback.
*   **Metrics Quantification:** Utilized core formulas (`COUNTIF`, `SUM`, `AVERAGE`) to establish baseline customer metrics and Average Order Value (AOV).

### 3. Stakeholder Management & Strategic Communications
Data insights are only valuable when aligned with corporate strategy and communicated clearly to decision-makers.
*   **Data-Driven vs. Data-Informed:** Balanced quantitative web funnel metrics with qualitative user context to build balanced business proposals.
*   **Stakeholder Alignment Matrix:** Designed reports tailored for the **Product Manager** (addressing technical cart abandonment) and the **Marketing Lead** (optimizing promotional strategy).
